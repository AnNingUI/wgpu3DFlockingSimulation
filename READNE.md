## Introduction
In Terminal 3D [Flock Simulation](https://www.red3d.com/cwr/papers/1987/SIGGRAPH87.pdf) with wgpu 
This project is only about rapid implementation, and there is no secondary optimization

## Attention
Run the final compiled executable in Terminal
After running, you can press `Esc` or `Q` to exit

## Compilation
```bash
git clone git@github.com:AnNingUI/wgpu3DFlockingSimulation.git
# cargo run can be run directly
# cargo build compilation
cargo build --release
```
